#1.Create a simple program that uses a while loop to iterate over a range of numbers from 1 to 10.
#2.Implement a condition to print only the even numbers within this range using the continue statement.
#3.Add another condition that stops the loop when it encounters the number 8, utilizing the break statement.
#4.Ensure your program includes comments explaining the purpose of each segment, especially where break and continue are used.
#5.Execute your program and document the output to verify it meets the expected behavior.

# Start from 1
i = 1

# Loop until 10
while i <= 10:

    # Skip odd numbers
    if i % 2 != 0:
        i += 1      # Increment before continue
        continue

    # Print even numbers
    print(i)

    # Stop when 8 is reached
    if i == 8:
        break

    # Move to next number
    i += 1
